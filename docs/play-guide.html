How to play

What is "Queen's game"?

Queen's game is origininated from Japanese party game, which is called "King's game(王様ゲーム)". This game is a sort of combination of, "Simon Says" and "truth or dare", but might be more flexible and thralling. We thought it would be great to introduce this game into a mobile application so that all over the people in world can know, play and share its wonderful experience!

This game simply consists of two factors, command and obey.

Participants decide on one "Queen" and others(citizens). Every citizen is provided a secret index that only him/herself knows.

Then, the Queen makes a command involving indexes, such as "No. 2 have to make No.3 laugh! 🤣".

Finally, each person reveals their index (number), and those who are targeted have to follow the command!

Game usage

Once you launch the app, users can select whether to start a new game or modifying game settings

New Game

The basic flow of the game is as follows.

1. Select the number of players

2. Enter user names

3. Choose the queen randomly

4. The queen chooses the command

5. Targets described in the command will be chosen randomly

6. Execute(Obey) the command
